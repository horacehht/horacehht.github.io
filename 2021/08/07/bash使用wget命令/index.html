Windows 10 git bash wget command not found

因为我之前在git bash中试过敲Linux的命令，比如ls,cd,mkdir

然后我搭服务器的时候就有用过wget来进行下载和tar来进行解压

我就想着git bash应该也可以运行其他的Linux命令吧。

于是敲了一下，报错wget command not found

我就在网上查找资料，最后成功实现了git bash中运用wget命令

步骤如下：

1.进入该网址https://eternallybored.org/misc/wget/，根据你电脑的位数（32，64）来下载对应wget相关的exe

（这里我下载了1.21.1版本的64-bit的exe）

2.将该exe文件移至git目录下的mingw64的bin文件夹下

我的是F:\Git\mingw64\bin，这个因人而异

3.将上面那个路径添加进环境变量

之后就可以快乐地在git bash里敲wget命令啦！

其他用不了的Linux命令也是这样解决
