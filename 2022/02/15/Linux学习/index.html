Linux文件系统

windows下有C，D盘的盘符区别，Linux没有盘符概念，只有一个根目录，以/为标识

根目录下有这几个目录：/bin,/etc,/home,/lib,/usr

作用详见：https://www.cnblogs.com/duanji/p/yueding2.html

/home/用户名：存储用户相关的一些文档

/usr：usr是user的缩写，安装的应用程序

命令

命令的格式为：cmd-option parameter

cmd是命令，就是一个操作，必须要有

parameter一般是要操作的对象

option一般是用来修饰parameter的，用于表明parameter是怎样的对象

cmd --help即可获得命令cmd的文档，介绍了命令的用法和作用，了解各参数的意义

man rm：man是manual的缩写，可以打开rm的用户手册，用上箭头和下箭头进行移动，退出按q

pwd：（print work directory）可查看当前所在目录

ls：（list）列出当前所有的文件，蓝色的代表文件夹，白色的代表文件

cd：（change directory）切换目录，

clear：清空整个终端界面的输出内容

cd

cd ..：切换到上一级目录，也可跳两级，要用左斜杠/
cd .：则是当前目录

cd ../..：连跳两级目录

cd ~：回到home目录

cd -：切换回上次操作所在的目录

ls

ls -l：以列表的形式列出（可以缩写成ll）

  • 可以看到文件创建的时间
  • 数字是目录/文件所占的字节大小
  • total是目录下的文件占的总大小（单位为kB）

但有隐藏的文件，那么怎么通过ls看到隐藏的文件呢？

ls -a：a是all的缩写，显示所有文件（包括隐藏文件）

-a和-l可以一起使用，而-h要和-l一起使用才能起作用，不同在于显示的单位转换成了K

不同的参数可以写在一块，比如说-lh，对顺序不敏感

ls -lha，其中有两个特殊的地方，出现了.，代表当前目录，出现也了..，代表上级目录，之所以出现了这两个，是需要记录文件夹的层级关系

我们发现隐藏文件都是以.开头的

通配符

缩小范围进行ls查找

*，匹配零个或多个任意字符，ls *.txt，则要求只列出以txt结尾的文件

可以这么理解，ls后面可跟正则表达式

?，匹配一个字符，ls ???.odt，只列出三个字母或数字的odt文件

可以根据*？组合出不同的正则表达式来进行文件的查询

[]，匹配中括号内任意一个字符，[1234]，匹配1或者2或者3或者4 ，也可以[1-4]匹配1到4内的任意一个数字

[ar1]，匹配三个字符中的任意一个，[a-z]匹配任意小写字符，然而大写的字母并不行

touch

用于创建文件，不能创建目录，touch 文件名.文件格式

mkdir

用于创建目录，mkdir 目录名，只能一级一级的创建目录

rm

remove的缩写，用于删除文件和目录

rm test.txt：删除test.txt文件

删除目录时需要添加一个参数-d，d是directory的意思，rm -d bb删除一个bb的目录

mv

move的缩写，用于移动文件

mv 123.txt bb：把123.txt文件移动到bb目录下，这是基于相对路径进行移动

mv 123.txt /home/用户名/Desktop/bb：把123.txt文件移动到桌面的bb目录下，这基于绝对路径进行移动

也可以移动目录，mv bb aa把bb目录移动到aa目录

cp

copy的缩写，用于复制文件和目录

cp a23.txt bb：把a23.txt拷贝到bb目录下

cp -r bb/aa .：把bb目录下的aa目录拷贝到当前目录（移动目录时需要指定-r）

cp a23.txt bb/a29.txt：把a23.txt拷贝到bb目录下，并重命名为a29

which

每个命令执行的时候，都会去执行一个程序。而which命令可以查看到命令程序所处的目录

命令一般都放到/bin中，bin是binary二进制的意思

但which cd是空的，这是因为cd是shell内置的命令

带s的命令，则需要更高的权限

find

具有文件搜索的功能，功能更强，速度受计算机影响

精确搜索

格式：find 搜索目录 搜索条件

比如说要在/home目录下，搜索一个125.odt文件，此时需要加一个-name选项

模糊搜索

此时搜索条件中加上通配符即可，即写成正则表达式。

find /home 12*

Linux中查找对大小写敏感，所以如果要忽略大小写，加一个-iname选项

按文件大小搜索

find 路径 -size +1，其中数字的单位为一个数据块，一个数据块为512k，而+号表示要搜索大于1数据块的文件/目录，-号则是小于

按文件所属人

find 路径 -user 用户名

按文件被修改时间

假如说电脑被黑客入侵了，知道大概的入侵时间，想知道什么文件被访问过了，就可以按文件修改时间来进行搜索

find 路径 -mmin -5 ，第一个m是modified的缩写，min说明单位是分钟，-号代表想查询小于5分钟之内的

mmin是文件内容被修改的时间

amin选项用来查询上次被访问的时间

cmin是文件属性（如权限）被修改的时间

按照文件类型

find 目录 -type 字母

根据字母的不同，可以查询不同的文件类型

f：（file）文件

d：（directory）目录

l：（link）软链接

搜索条件复合

利用-a连接两个条件达到and效果，连接的条件都需要符合

-o连接两个条件达到or效果，连接的条件符合一个即可

根据id进行搜索

id是不重复的，首先我们要知道文件的id，此时我们可以用ls -i显示文件的id

find 路径 -inum id，inum是id number

cat

用于查看文件内容，cat 123.txt，查看123.txt文件内容

-b：输出时文件内容时标出行号，空行不计入行数计算

-n：与-b类似，但空行计入行数计算

more

用于查看文件内容，与cat不同的点在于more分页展示。

grep

用于文件内容的搜索，会显示出文件中包含关键字的那行内容

gerp user 123.txt：在123.txt中查询user行的内容

-n：搜索出来时标出搜索内容所在的行

-i：搜索时忽略大小写

-v：反向搜索，搜索不包含关键字的内容，如grep '#' 123.txt，在123.txt中输出不含#号的内容（即去掉注释）

如果我们只想去掉以#开头的那部分内容，那就在#号前加上一个**尖号^**，表示以什么开头，如gerp ^'#' 123.txt

如果只搜以s结尾的，则在内容后加上一个**美元符号$**，如gerp s$ 123.txt，表示从123.txt中搜索以s结尾的行

echo

回显命令

echo hello，则会在控制台上输出hello

还可以将内容写入一些文件中，如echo hello > 123.txt，把hello写入123.txt中，但会覆盖掉原文件内容

如果我们只是想追加内容到文件中，不覆盖原文件，我们把>改成>>即可，echo hello >> 123.txt

sudo

sudo可以获取到超管（root）权限，常加载一些需要超管权限的命令前

sed

用于文本内容的编辑

比如要在train.py文件的最后一行追加一行，则可以这样写

a是addition

tar

对文件进行打包压缩

ps

此命令用于显示当前进程的状态

chmod

用于更改文件权限

生成一个sh文件

接着在.sh文件中输入

此时控制台则可输出Hello world

+：增加文件的某个权限

-：去除文件的某个权限

读，写，执行权限分别对应r,w,x三个字母

wget

是一个文件下载工具，用于下载网络上的各种资源和软件

硬链接与软链接

Linux下的链接类似于windows下的快捷方式，Linux下分有两种链接方式：硬链接，软链接

硬链接

在Linux中，一个文件被创建时，文件也有其自身的inode（称为索引节点，相当于文件的地址），而硬链接是通过索引节点进行的链接。硬链接是创建一个具有相同inode但文件名不同的文件，所以删除硬链接对应的源文件，硬链接文件仍然存在，可以起到防误删的作用。

与复制操作的区别相当于：复制会生成一个inode不同的文件。

其通过ln或link命令创建，如ln oldfile newfile

软链接

也称符号链接，生成的软链接文件存放的内容是另一文件的路径的指向，可以对文件或目录创建。

软链接主要应用于以下两个方面：

一是方便管理，例如可以把一个复杂路径下的文件链接到一个简单路径下方便用户访问；

另一方面就是解决文件系统磁盘空间不足的情况。例如某个文件文件系统空间已经用完了，但是现在必须在该文件系统下创建一个新的目录并存储大量的文件，那么可以把另一个剩余空间较多的文件系统中的目录链接到该文件系统中，这样就可以很好的解决空间不足问题。删除软链接并不影响被指向的文件，但若被指向的原文件被删除，则相关软连接就变成了死链接。

ln -s 目标目录 软链接地址，如ln -s /var/www/test test，当前路径创建test，其指向/var/www/test目录

目标目录为软链接指向的目标目录，软链接地址为“快捷键”文件名称

删除软链接用rm -rf即可

修改软链接ln -snf 新目标目录 软链接地址

下面为演示样例，桌面下有bb目录，bb目录下有aa目录

此时我们在桌面下创建一个指向bb目录的软链接，命名为soft

关于用户

Linux系统是一个多用户多任务的分时操作系统，任何一个要使用系统资源的用户，都必须首先向系统管理员申请一个账号，然后以这个账号的身份进入系统。

用户的账号一方面可以帮助系统管理员对使用系统的用户进行跟踪，并控制他们对系统资源的访问；另一方面也可以帮助用户组织文件，并为用户提供安全性保护。

添加用户useradd

useradd [option] username

option

-g：指定用户所属的用户组

如-g group1，则该用户属于group1组

-d：指定用户主目录，若该目录不存在 ，则加上-m选项

useradd -d /home/sam -m sam，创建一个用户sam，其主目录在/home/sam下

删除账号userdel

userdel [option] username

常用的选项是 -r，它的作用是把用户的主目录一起删除。

用户口令管理passwd

用户账号刚创建时账号被锁定，必须为其指定口令后方可使用

passwd [option] username

  • -l 锁定口令，即禁用账号。
  • -u 口令解锁。
  • -d 使账号无口令。
  • -f 强迫用户下次登录时修改口令。
